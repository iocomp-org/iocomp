Detailed Description

Contains function for stopping send of data to I/O server.

Contains function for stopping send of data to I/O server. For HT mode, sends a ghost message to the paired rank in the interComm. This ghost message is a signal to the I/O server to stop receiving data by sending a message with length 0. For shared mode, assigns win free to the wintestflags and calls MPI free For sequential mode, it finalises the adios2 object and prints the timers for I/O times.

void ioServer_shared ( struct iocomp_params *  iocompParams)

Allocates shared windows and their pointers.

Allocates shared windows and their pointers. Initialises IO server cartesian communicators, and ADIOS2 objects. Then starts the loop for the IO server, which starts with receiving the Broadcast control state array from the compute processes. The state of each window is checked with the control state array and the appropriate action is taken for each window.

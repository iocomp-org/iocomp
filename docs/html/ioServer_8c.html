Detailed Description

Contains functions for I/O server for HT flag and sequential mode.

Contains functions for I/O server for HT flag. For HT mode, initialises the server for loop which gets the length of the message using MPI_Probe. If message length is 0 i.e. a ghost message it exits the loop. Otherwise, it receives the message and writes it using the appropriate I/O library. If message length is same as previous length, it keeps the previous initialisations. After receiving the message, it recieves the file name and writes it under that file name.

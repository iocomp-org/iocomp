Detailed Description

Implements splitting of MPI communicator into compute and I/O communicators for shared memory.

Splits MPI communicator into compute and I/O communicators for shared memory. Makes globalsize/2 pairs of compute and I/O ranks such that a compute and I/O rank have a mutual communicator. Assuming IO process and Compute Process are mapped to physical and SMT cores if size = 10 then IO rank would be 5,6,..9 and compute rank would be 0,1,..4 Assign similar colours to corresponding I/O and compute Process i.e. rank 0 and rank 5 would have same colour and then same MPI Communicator.
